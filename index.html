Magazine Identification on SS1-V1 Assault Rifle using Web-based HX711 Load Cell Sensor

Published in INFOTEL Journal (SINTA 2), 2024 by Yasikha

This research explores a web-based monitoring system designed to detect the magazine status of SS1-V1 assault rifles using HX711 Load Cell sensors. The goal was to enhance weapon storage safety through an integrated real-time monitoring platform.

My contribution focused on developing the system architecture and implementing the web interface to visualize sensor data effectively. The system was tested in a simulated environment and showed consistent performance in identifying weight variations, enabling accurate magazine status detection.

This work was successfully published in INFOTEL, a nationally accredited scientific journal (SINTA 2), under the supervision of the COE STAS Research Group at Telkom University.
